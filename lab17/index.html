OOP Hangman

Understand the problem.

Set up the game board and players.
Game loop.
Establish the word that is being guessed.
Generate one empty space for each letter in the word.
Get letter guess from player.
If correct, fill empty space(s) with corresponding letter
If incorrect, add letter to incorrect box and generate a body part on the hanged man.
If all body parts generated, end game.
Prompt player for word guess.
If correct, end game.
If incorrect, start the process over.
